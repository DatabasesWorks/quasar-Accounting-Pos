Terms List



The terms list displays a list of the payment terms defined for both vendors and customers.

List Table

  • Name - The name of the term.
  • Inactive - The active state of the term.

Show Inactive

Toggle on or off. Toggle on to display both active and non-active terms. Toggle off to display only active terms.

New

Select this function to create a term.

Edit

Select this function to edit the high-lighted term.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
