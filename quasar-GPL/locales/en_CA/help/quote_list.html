Quote List



The quote list displays the complete list of customer quotations. New quotations may be created and existing quotations may be viewed and edited. All customer quotes can be quickly turned into a customer invoice. The quote list may be sorted by any of the headers in the table.

List Table

  • Quote - The identification number of the quote.
  • Date - The date the quote was created.
  • Customer - The customer the quote was created for.
  • Amount - The total amount of the quote.

Show Inactive?

Toggle on or off. Toggle on to display both active and non-active quotes Toggle off to display only active quotes.

New

Select this function to create a new quote.

Edit

Select this function to edit the high-lighted quote.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
