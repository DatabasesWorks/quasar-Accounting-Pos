Extra Info

Each company can add attributes and store information required specifically by that company.

  • Data Name - Displays the name of defined attributes.
  • Value - Enter the data value specific to this personal acquaintance.
  • Create Data - Click on the "Create Data" button to create a new data attribute that will be displayed for all personal acquaintances.
  • Delete Data - Click on the "Delete Data" button to delete the high-lighted data attribute. BEWARE: This will delete the attribute for all personal acquaintances and all data values will be lost.
  • Rename Data - Click on the "Rename Data" button to rename the high-lighted data attribute.

Email

Click on the email button to send an email message.
