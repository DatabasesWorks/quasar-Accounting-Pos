Tender List



The tender list displays a complete list of tenders. New tenders may created and existing tenders may be viewed and edited.

List Table

Name

The name of the tender.

Type

There are four tender types. These include cash, cheque, card and charge.

Inactive

The active state of the tender.

Show Active

Toggle on or off. Toggle on to display both active and non-active tenders. Toggle off to display only active tenders.

New

Select this function to create a tender.

Edit

Select this function to edit the high-lighted tender.
