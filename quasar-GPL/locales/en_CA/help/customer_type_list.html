Customer Type List



The customer type list displays a list of the various customer type records that have been created. Customer types can be used to define default information that will be used when creating new customer records.

List Table

  • Name - The name of the customer type.
  • Account - The ledger account that the type is linked to.
  • Inactive - The active state of the customer type.

Show Inactive

Toggle on or off. Toggle on to display both active and non-active customer types. Toggle off to display only active customer types.

New

Select this function to create a customer type.

Edit

Select this function to edit the high-lighted customer type.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
