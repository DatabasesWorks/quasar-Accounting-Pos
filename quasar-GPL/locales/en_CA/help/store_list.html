Store List



The store list displays a list of physical stores within the company.

List Table

  • Name - The name of the store.
  • Number - The number assigned to the store.
  • Sell? - Displays whether or not the company sells products or services at the named stores.

New

Select this function to create a new store.

Edit

Select this function to edit the high-lighted store.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
