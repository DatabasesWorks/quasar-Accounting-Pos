Order Templates List



The order template list displays a list of purchase order templates. These purchase order templates can be used by Quasar to create an automatic order for the items that exist on the template.

List Table

  • Name - The name of the order template.
  • Created The date that the order template was created. When two order templates have the same date, you can tell the most current by the date.
  • Inactive - The active state of the order template.

Show Inactive

Toggle on or off. Toggle on to display both active and non-active order templates. Toggle off to display only active order templates.

New

Select this function to create an order template.

Edit

Select this function to edit the high-lighted order template.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
