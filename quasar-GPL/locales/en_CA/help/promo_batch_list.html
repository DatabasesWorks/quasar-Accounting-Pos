Promo Batch List



The promo batch list displays the batch number and description of a promotion batch.

List Table

  • Number - The promotion batch number
  • Description - The description of the promotion batch
  • Inactive? - The active state of the promotion. NOTE: by setting a promotion batch to inactive Quasar sets all the price records for the promotion to inactive automatically.

Show Inactive?

Toggle on or off. Toggle on to display both active and non-active batches. Toggle off to display only active batches.

New

Select this function to create a new promotion batch.

Edit

Select this function to edit the high-lighted batch.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
