User List



The user list displays a complete list of users and their security type. New users may created and existing users may be viewed and edited.

List Table

  • Name - The name of the user
  • Security - A security type that the user is linked to.
  • Inactive - The active state of the user.

Show Inactive?

Toggle on or off. Toggle on to display both active and non-active users. Toggle off to display only active users.

New

Select this function to create a user.

Edit

Select this function to edit the high-lighted user.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
