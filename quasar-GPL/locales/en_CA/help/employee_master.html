Employee Master

Company (toggle on)

Toggle on or off. Toggle on if the employee is a company.




Company Name

The name of the company.

Contact

The contact name for the company.

Number

Enter a number that can be used to identify an employee. For example, when entering an employee on a customer invoice, purchase order or vendor invoice you can enter the number rather than the employee name and the employee name will be displayed.

Company (toggle off)

Toggle on or off. Toggle off if the employee is an individual person.




Last Name

The employee's surname.

First Name

The employee's given name.

Number

Enter a number that can be used to identify an employee. For example, when entering an employee on a customer invoice or vendor invoice you can enter the number rather than the employee name. The employee name will be displayed.

Address

Street

The employee's street address.

City

The city in which the employee resides or carries out business.

Province

The province in which the employee resides or carries out business.

Country

The country in which the employee resides or carries out business.

Postal Code

The employee's postal code.

Phone #

The employee's phone number.

Phone2 #

The employee's alternate phone number.

Fax

The employee's fax number.

Email

The employee's email address.

Web Page

The employee's web page address.

Point-of-Sale

Enter information specific to the employee acting as a cashier at the point-of-sale.

  • Password - Enter the password that the employee will use when they login as at cashier at the point-of-sale.
  • Security Level - Select the security level that the employee will have at the point-of-sale.
    • No Access - Select "No Access" if the employee will not have access to the point-of-sale.
    • Cashier - Select "Cashier" if the employee will have cashier level security at the point-of-sale. This is the lowest level of security access.
    • Manager - Select "Manager" if the employee will have manager level security at the point-of-sale. This is the second level of security access.
    • Supervisor - Select "Supervisor" if the employee will have supervisor level security at the point-of-sale. This is the highest level of security access.

Groups

Enter a group that links this employee to other employees in the same group.

Extra Info

Each company can add attributes and store information required specifically by that company.

  • Data Name - Displays the name of defined attributes.
  • Value - Enter the data value specific to this employee.
  • Create Data - Click on the "Create Data" button to create a new data attribute that will be displayed for all employees.
  • Delete Data - Click on the "Delete Data" button to delete the high-lighted data attribute. BEWARE: This will delete the attribute for all employees and all data values will be lost.
  • Rename Data - Click on the "Rename Data" button to rename the high-lighted data attribute.

Inactive?

Toggle on or off. Toggle on if the employee is no longer active. Toggle on to make the employee active again.

Email

Click on the email button to send an email message.
