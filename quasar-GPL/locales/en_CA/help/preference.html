Table of Contents Quasar Commands

Preferences

The preference window can be accessed by selecting "File" from the menu bar on the main window. Then by selecting "Preferences".

Display

Select "Display" to define the look and feel of the interface that you prefer to work with.




Change Style?

Toggle on or off. Toggle on to change the style. Toggle off to the default style. Select the CDE, Motif, Platinum or Windows style.




Change Color?

Toggle on or off. Toggle on to change the color. Toggle off to set to the default color. Use the color chart to select new colors




Change Font?

Toggle on or off. Toggle on to change the font style and/or the font size. Toggle off to set to the default font. Use the font chart to select new font styles and font size




Date Format

Define the date format that you prefer to work with.




Date Order

Select the order that the date will be displayed. Select from day/month/year, month/day/year or year/month/date.




Separator

Enter the character that will separate the day, month and year. For example you could use a comma to have your date look like this "21,December,2000"

Numeric

Toggle on or off. Toggle on to set the date format to numeric only. Toggle off to set the date format to alpha-numeric.

Example

Displays an example of your current date format.

OK

Select "OK" to immediately apply your changes and to save your changes for future sessions.

Apply

Select "Apply" to immediately apply your changes to your current session. Your changes will not be saved for future sessions.

Defaults

Select "Defaults" to change the preferences back to the default settings.
