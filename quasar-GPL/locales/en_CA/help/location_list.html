Location List



The location list displays a complete list of the inventory locations specified for your company. You may specify one location, per item, per store. Defining the location or address of your items is very helpful when running reports and/or doing functions like taking physical inventory.

List Table

  • Name - The name of the location. Location names can include up to three parts seperated by commas. In the example "Tools,Aisle1,34" the item can be found in the tool section, in aisle1, bin 34.
  • Inactive - The active state of the location..

Show Inactive

Toggle on or off. Toggle on to display both active and non-active locations. Toggle off to display only active locations.

New

Select this function to create a new location.

Edit

Select this function to edit the high-lighted location.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
