Change Password



The change password is used to change a users login password. Only the user logged in can change their password using this screen.

Current Password

Enter your current password. This is a security feature so someone just walking by your computer cannot change your password.

New Password

Enter your new password.

Retype to Verify

Verify your new password by retyping it.
