Adjustment Reason List



The adjustment reason list displays a complete list of reasons for making inventory adjustments. New adjustment reasons may be created and existing adjustment reasons may be edited or deleted.

List Table

  • Name - The name of the inventory adjustment reason.
  • Number - The inventory adjustment reason.
  • Inactive - The active state of the adjustment reason.

Show Inactive

Toggle on or off. Toggle on to display both active and non-active inventory adjustment reasons. Toggle off to display only active reasons.

New

Select this function to create a new adjustment reason.

Edit

Select this function to edit the high-lighted adjustment reason.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
