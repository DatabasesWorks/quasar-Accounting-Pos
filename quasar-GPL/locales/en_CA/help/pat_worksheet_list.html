Patronage Worksheet List



The patronage worksheet list displays a list of patronage worksheets. Patronage worksheets are created to view and to allocate patronage equity or credits.

List Table

  • Description - The description of the patronage worksheet.
  • Inactive - The active state of the customer type.

Show Inactive

Toggle on or off. Toggle on to display both active and non-active patronage worksheets. Toggle off to display only active patronage worksheets.

New

Select this function to create a patronage worksheet.

Edit

Select this function to edit the high-lighted patronage worksheets.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
