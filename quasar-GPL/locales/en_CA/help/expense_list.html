Expense List



The expense list displays a complete list of expenses that may be used in a pay out at a Quasar Point-of-Sale terminal. New expenses may created and existing expenses may be viewed and edited.

List Table

Name

The name of the expense.

Number

The number of the expense.

Inactive

The active state of the expense.

Show Active

Toggle on or off. Toggle on to display both active and non-active expenses. Toggle off to display only active expenses.

New

Select this function to create a expense.

Edit

Select this function to edit the high-lighted expense.
