Shift List



The shift close list displays the complete list of shift close transactions.

List Table

  • Number - The identification number of the shift close transaction.
  • Date - The date the shift close was created.
  • Station - The work station or point-of-sale device for which the shift close was creted.
  • Employee - The employee for which the shift close was created.

Show Inactive?

Toggle on or off. Toggle on to display both active and inactive shift closes. Toggle off to display only active shift closes.

New

Select this function to create a new shift close.

Edit

Select this function to edit the high-lighted shift close.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
