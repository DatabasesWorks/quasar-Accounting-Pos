Tax List



The tax list displays a complete list of taxes and the tax percentages. New taxes may created and existing taxes may be viewed and edited.

List Table

  • Name - The name of the tax.
  • Description - The description of the tax.
  • Rate - The percentage rate of the tax.
  • Inactive - The active state of the tax.

Show Inactive?

Toggle on or off. Toggle on to display both active and non-active taxes. Toggle off to display only active taxes.

New

Select this function to create a tax.

Edit

Select this function to edit the high-lighted tax.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
