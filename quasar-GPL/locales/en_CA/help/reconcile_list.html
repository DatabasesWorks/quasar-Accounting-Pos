Reconcile List



The reconcile list displays a complete list of reconciliations. By default, only incomplete reconciliations will display upon entry. To display and/or reprint completed reconciliations click on the "Show Reconciled?" toggle.

List Table

  • Account - The ledger account being reconciled.
  • Date - The date of the account reconciliation.
  • Reconciled? - Shows which reconciliations have been completed (reconciled).

Show Reconciled?

Toggle on or off. Toggle on to display both reconciled (completed) and non-reconciled reconciliations. Toggle off to display only current reconciliations.

New

Select this function to create a new account reconciliation.

Edit

Select this function to edit the high-lighted reconciliation.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
