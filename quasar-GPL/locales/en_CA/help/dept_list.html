Department List



The department list displays a complete list of inventory departments. New deparments may created and existing departments may be viewed and edited.

List Table

  • Name - The name of the department
  • Number - The department number
  • Inactive - The active state of the sub-department

Show Inactive

Toggle on or off. Toggle on to display both active and non-active departments. Toggle off to display only active departments.

New

Select this function to create a new department.

Edit

Select this function to edit the high-lighted department.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
