Discount List



There is an audit trail for each discount given in Quasar. Whenever a price discount is given a pre-defined discount name must be linked to it.

List Table

  • Name - The name of the price discount.
  • Line - Displays if the discount is a line discount.
  • TX? - >Displays if the discount is a transaction discount.
  • Method - Identifies the discount method. A discount is either a percent discount or a dollar discount.
  • Amount Displays a percent discount as a percentage and displays a dollar discount in dollars.
  • Inactive - The active state of the discount.

Show Inactive

Toggle on or off. Toggle on to display both active and non-active discounts. Toggle off to display only active discounts.

New

Select this function to create a discount.

Edit

Select this function to edit the high-lighted discount.

Refresh

Clicking on refresh will refresh the data on the screen.
