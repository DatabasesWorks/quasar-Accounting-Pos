Patronage Group List



The patronage group list displays a list of patronage groups. Patronage groups are created to link customer accounts for the purpose of allocating patronage.

List Table

  • Name - The name of the patronage groups.
  • Number - The number assigned to patronage groups.
  • Inactive - The active state of the patronage group.

Show Inactive

Toggle on or off. Toggle on to display both active and non-active patronage worksheets. Toggle off to display only active patronage groups.

New

Select this function to create a patronage group.

Edit

Select this function to edit the high-lighted patronage groups.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
