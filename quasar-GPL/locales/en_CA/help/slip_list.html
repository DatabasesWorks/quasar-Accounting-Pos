Receiving List



The receiving list contains a list of all receivings. A receiving is nothing more than a count of items received.

List Table

  • Number - The identification number of the receiving.
  • Date - The date the receiving was created.
  • Vendor - The card identification of the vendor.
  • Closed? - The state of the receiving. Once a receiving is allocated to a vendor invoice it is automatically closed.

Show Closed?

Toggle on or off. Toggle on to display both active and closed receivings. Toggle off to display only open receivings.

New

Select this function to create a new receiving.

Edit

Select this function to edit the highlighted receiving.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
