To Do List



The to do list displays a complete list of things that must be done and the date they must be done by. New todo notes may created and existing todo notes may be viewed and edited.

List Table

  • Description - The description of the todo note. Actually this taken from the very first line of the todo note. Therefore, it is important to enter a good description as the first part of your todo note.
  • Remind On - The date you are reminded to do the task.
  • Done - Displays if the todo task is done.

Show Done?

Toggle on or off. Toggle on to show todo tasks that have been done. Toggle off to hide done tasks.

New

Select this function to create a todo note.

Edit

Select this function to edit the high-lighted todo note.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
