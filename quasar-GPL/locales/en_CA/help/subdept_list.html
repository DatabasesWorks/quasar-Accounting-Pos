Sub-Department List



The sub-department list displays a complete list of inventory sub-departments. New sub-deparments may created and existing sub-departments may be viewed and edited.

List Table

  • Name - The name of the sub-department
  • Number - The number assigned to the sub-department
  • Department - The department that the sub-department is linked to.
  • Inactive - The active state of the sub-department.

Show Inactive

Toggle on or off. Toggle on to display both active and non-active sub-departments. Toggle off to display only active sub-departments.

New

Select this function to create a new sub-department.

Edit

Select this function to edit the high-lighted sub-department.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
