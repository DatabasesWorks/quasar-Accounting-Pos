Security Type List



The security type list displays the various security types that have been created to control access to the company data. Only those uses with proper access can create, view, edit and delete records.

List Table

  • Name - The name of the security type.

New

Select this function to create a new security record.

Edit

Select this function to edit the high-lighted security record.

Refresh

Clicking on refresh will refresh the data on the screen.

Print

Select this function to print the contents of the list to the printer.
